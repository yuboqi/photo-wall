# 📸 拍立得照片墙

一个有趣的在线拍立得照片墙应用，可以通过摄像头拍照或上传本地照片，生成拍立得风格的照片，并在照片墙上自由摆放。

## ✨ 功能特点

### 📷 拍照功能
- **摄像头拍照**：点击"打开摄像头"按钮，可以使用电脑或手机的摄像头进行拍照
- **照片预览**：拍照后会显示预览，确认满意后照片会自动添加到照片墙
- **摄像头控制**：可以随时打开或关闭摄像头

### 📁 上传功能
- **本地上传**：点击"选择照片"按钮，可以从电脑或手机中选择照片上传
- **支持格式**：支持所有常见的图片格式（JPG、PNG、GIF等）

### 🖼️ 相纸样式
提供多种拍立得相纸样式供选择：
1. **经典白**：经典的白色拍立得相纸
2. **复古黄**：复古怀旧的黄色相纸
3. **现代黑**：时尚的黑色相纸
4. **自定义**：可以上传自己的相纸图片作为边框

### 🎨 背景选择
提供多种照片墙背景：
1. **木纹**：温暖的木质纹理背景
2. **墙面**：简约的墙面纹理背景
3. **布纹**：柔和的布料纹理背景
4. **自定义**：可以上传自己喜欢的背景图片

### 🎯 照片操作
- **自由拖拽**：每张照片都可以用鼠标或手指拖动到任意位置
- **随机摆放**：新添加的照片会随机摆放在照片墙上，并带有随机的旋转角度
- **悬停控制**：鼠标悬停在照片上会显示操作按钮
  - 💾 **保存按钮**：保存单张拍立得照片（包含相纸边框）
  - 🗑️ **删除按钮**：删除这张照片

### 💾 导出功能
- **保存单张照片**：点击照片上的保存按钮，可以将单张拍立得照片（包含相纸边框）保存到本地
- **保存整个照片墙**：点击右上角的"保存照片墙"按钮，可以将整个照片墙保存为一张大图

## 🚀 使用方法

### 快速开始
1. 双击打开 `index.html` 文件，或者在浏览器中打开
2. 不需要安装任何软件，直接在浏览器中使用

### 拍照步骤
1. 点击左侧的"打开摄像头"按钮
2. 浏览器会请求摄像头权限，点击"允许"
3. 调整好角度后，点击"拍照"按钮
4. 照片会自动以拍立得的形式出现在右侧照片墙上

### 上传照片步骤
1. 点击左侧的"选择照片"按钮
2. 从电脑或手机中选择一张照片
3. 照片会自动以拍立得的形式出现在右侧照片墙上

### 调整照片位置
1. 用鼠标点击并拖动照片到想要的位置
2. 在手机上可以用手指拖动照片
3. 松开鼠标或手指，照片就会固定在新位置

### 更换相纸样式
1. 在左侧"选择相纸样式"区域点击想要的样式
2. 之后添加的照片都会使用新选择的相纸样式
3. 点击"自定义"可以上传自己的相纸图片

### 更换背景
1. 在左侧"选择背景"区域点击想要的背景
2. 照片墙的背景会立即更换
3. 点击"自定义"可以上传自己的背景图片

### 保存照片
1. **保存单张照片**：
   - 将鼠标悬停在想要保存的照片上
   - 点击右上角的💾按钮
   - 照片会自动下载到电脑的下载文件夹
   
2. **保存整个照片墙**：
   - 点击右上角的"💾 保存照片墙"按钮
   - 整个照片墙会被保存为一张大图
   - 图片会自动下载到电脑的下载文件夹

### 删除照片
1. 将鼠标悬停在想要删除的照片上
2. 点击右上角的🗑️按钮
3. 确认删除后，照片会从照片墙上移除

## 💡 使用技巧

1. **摄像头权限**：首次使用摄像头时，浏览器会请求权限，请点击"允许"
2. **照片质量**：导出的照片使用2倍分辨率，确保清晰度
3. **照片数量**：可以添加任意数量的照片到照片墙
4. **浏览器兼容**：建议使用Chrome、Edge、Safari等现代浏览器
5. **手机使用**：在手机上也可以正常使用，支持触摸拖动

## 🛠️ 技术实现

### 技术栈
- **HTML5**：页面结构
- **CSS3**：样式和动画效果
- **JavaScript**：交互逻辑
- **html2canvas**：用于将照片墙导出为图片

### 核心功能实现
1. **摄像头调用**：使用 `navigator.mediaDevices.getUserMedia()` API
2. **图片处理**：使用 Canvas API 进行图片捕获和处理
3. **拖拽功能**：使用原生的鼠标和触摸事件实现拖拽
4. **图片导出**：使用 html2canvas 库将DOM元素转换为图片

## 🎨 设计特点

1. **拍立得风格**：照片采用经典的拍立得相纸设计，底部留白更大
2. **随机摆放**：照片会随机旋转-10°到10°，更有真实感
3. **响应式设计**：在电脑和手机上都能良好显示
4. **流畅动画**：所有交互都有平滑的过渡动画
5. **直观操作**：所有功能都有清晰的图标和提示

## ⚠️ 注意事项

1. **浏览器兼容性**：
   - 摄像头功能需要浏览器支持 `getUserMedia` API
   - 建议使用最新版本的Chrome、Edge、Safari、Firefox浏览器
   - IE浏览器不支持

2. **权限要求**：
   - 使用摄像头需要授予浏览器摄像头权限
   - 如果拒绝了权限，可以在浏览器设置中重新开启

3. **HTTPS要求**：
   - 在某些浏览器中，摄像头功能只能在HTTPS或localhost下使用
   - 如果打开本地文件无法使用摄像头，可以使用本地服务器

4. **性能考虑**：
   - 照片数量过多时可能影响性能
   - 建议单个照片墙不超过50张照片

## 🔧 常见问题

**Q: 摄像头无法打开怎么办？**
A: 请检查：
- 浏览器是否有摄像头权限
- 摄像头是否被其他应用占用
- 是否使用了支持的浏览器

**Q: 如何在手机上使用？**
A: 直接在手机浏览器中打开 `index.html` 文件，所有功能都支持触摸操作。

**Q: 导出的图片在哪里？**
A: 图片会自动下载到浏览器的默认下载文件夹（通常是"下载"文件夹）。

**Q: 可以同时使用多个相纸样式吗？**
A: 每次只能选择一种相纸样式，但可以随时切换。已添加的照片会保持原有样式，新添加的照片使用新样式。

**Q: 自定义背景图片有什么要求？**
A: 建议使用分辨率较高的图片，图片会自动适应照片墙的大小。
